The FileOrganiser plugin allows you to organise your files virtually, i.e. independently of where they are physically located, remembering your virtual environment from one session to another. By default, your virtual environment will be empty:

Say that you are wokring on a particular modelling study. You might then want to create a (virtual) folder for your study and have your models (which could be physically located anywhere). To do so, you would first click on the icon in the toolbar (or by using the context menu). This will add a folder to your virtual environment:

You can rename the folder as you wish, as well as create other ones, including some sub-folders, if needed:

You can move the (sub-)folders around by simply dragging and dropping them, or delete an existing (sub-)folder by clicking on the icon in the toolbar (or by using the context menu):

Next, you might want to open the File Browser window, so you can start dragging and dropping files into your virtual environment:

As for folders, you can move and delete your (virtual) files:
