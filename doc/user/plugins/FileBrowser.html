The FileBrowser plugin offers a convenient way to access your physical files, remembering the folder or file which was selected when you last ran OpenCOR. By default, it will start by selecting your home directory:

As expected, to double click on a folder will expand its contents, as can be seen by expanding the Windows directory:

While double clicking on a file will result in it being opened by OpenCOR. The rendering of the file will depend on the current view plugin. In the case of the CellMLAnnotationView plugin, it will look as follows:

Folders and files can also be dragged from the File Browser window before being dropped onto the File Organiser window.
