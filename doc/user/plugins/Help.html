In addition to what is shown on the website, the Help plugin also shows special links which when clicked send a command to OpenCOR, asking it to perform a particular task. For example, open the current page both in OpenCOR and on the OpenCOR website. To open the About box, you can either select the Help | About... menu or click here...

The Help window also comes with a toolbar which offers the following features:

Go to the home page
Go back
Go forward
Copy the selection to the clipboard
Reset the size of the help page contents
Zoom in the help page contents
Zoom out the help page contents
Print the help page contents
