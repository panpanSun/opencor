The best way to familiarise yourself with OpenCOR's internals is by having a look at its source code.

Namespace

All of OpenCOR's code is and should be made available under the OpenCOR namespace (e.g. [OpenCOR]/src/mainwindow.cpp and [OpenCOR]/src/mainwindow.h). The only exception to this rule is [OpenCOR]/src/main.cpp.
